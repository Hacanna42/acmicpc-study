- 아래 코드에서, model이 `hello.html`에 전달되는 과정을 코드로 관찰할 수 없는데 어떻게 전달되는 것인가?

```java
@GetMapping("/hello")  
public String world(@RequestParam(name = "name", required = false, defaultValue = "World") String name, Model model) {  
    model.addAttribute("name", name);  
    return "hello";  
}
```
