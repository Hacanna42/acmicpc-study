## 버그 및 UX 수정 수십개
* 기존의 모든 기능이 업데이트 이후에도 정상 동작하는지 확인해주세요.

* 커뮤니티 프로필 화면에서 이모지가 중앙에 정렬되어 보이는지 확인해주세요.

* 꿈 일기 패턴 분석을 요청한 직후, 분석이 아직 완료되지 않았는데 “분석이 준비되었어요” 같은 안내가 잘못 표시되지 않는지 확인해주세요.

* 꿈 패턴 분석 완료 알림을 눌렀을 때, 바로 분석 화면으로 이동하는지 확인해주세요.

* 댓글에 답글이 달렸을 때, 댓글 작성자뿐 아니라 원글 작성자에게도 알림이 도착하는지 확인해주세요.

* 앱 언어가 한국어일 때 구독 가격이 달러가 아닌 올바른 지역/언어 기준으로 표시되는지 확인해주세요.

* 꿈 별자리 일괄 생성 완료 후 알림이 정상적으로 도착하는지 확인해주세요.

* 온보딩 화면에서 이미 알림 권한을 허용한 경우, “알림 허용하기” 버튼이 어색하지 않게 표시되는지 확인해주세요.

* 언어 설정이 시스템 값일 때 언어 선택 화면의 표시와 체크 상태가 자연스럽게 보이는지 확인해주세요.

* 회원가입 텍스트 버튼이 더 쉽게 눌리는지 확인해주세요.

* 회원가입 또는 로그인 화면에서 키보드의 “확인” 버튼을 눌렀을 때 다음 입력칸으로 자연스럽게 이동하는지 확인해주세요.

* 비밀번호 입력 시 스페이스바가 입력되지 않는지 확인해주세요.

* 태그 입력 후 포커스가 벗어나면 작성 중이던 태그가 자동으로 태그 UI로 전환되는지 확인해주세요.

* 사진 첨부 중인 이미지를 길게 눌러 순서를 변경할 수 있는지 확인해주세요.

* 커뮤니티 화면에서 빠르게 스크롤해도 로딩이 지나치게 늦거나 끊겨 보이지 않는지 확인해주세요.

* 커뮤니티 리스트를 가장 아래까지 내렸을 때 불필요한 공백이 남지 않는지 확인해주세요.

* 프로필 미리보기 화면에서 이메일이 노출되지 않는지 확인해주세요.

* 커뮤니티 북마크 기능이 정상적으로 동작하는지 확인해주세요.

* 일기장과 커뮤니티 등 여러 화면에서 소제목 스타일이 일관되게 보이는지 확인해주세요.

* 태그 삭제 x 버튼이 더 쉽게 눌리는지 확인해주세요.

* 사진을 4장 이상 첨부할 때도 정상적으로 추가되는지 확인해주세요.

* 글 작성/수정 화면에서 등장인물과 장소 입력도 포커스가 벗어나면 자동으로 라벨 형태로 전환되는지 확인해주세요.

* 앱 업데이트 후 로그인 상태는 유지되지만 프로필 탭의 닉네임, 꿈 경험치, AI 크레딧, 이메일 인증 여부, 아이콘, 이메일 등이 비어 보이지 않는지 확인해주세요.

* 앱 언어를 영어로 설정한 상태에서 한국어로 일기를 작성했을 때, 제목 추천 기능이 한국어 일기 내용에 맞게 동작하는지 확인해주세요.
