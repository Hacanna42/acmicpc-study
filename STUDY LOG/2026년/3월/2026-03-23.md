- Lume(Mac VM) 환경에 OpenClaw를 구축하여 기본적인 샌드박스 격리를 적용했다.

- 로컬 LLM은 Ollama를 통해 구성했으며, glm-4.7-flash, qwen3:30b 등 여러 모델을 비교 테스트한 결과 최종적으로는 glm-4.7-flash를 주력 모델로 선택했다.
  
- OpenClaw의 세부 설정 과정에서는 일부 구간에서 OpenAI Codex의 도움을 받아 환경을 정리했다.

- 또한 glm-4.7-flash의 temperature를 1.0에서 0.7로 낮춘 뒤, 응답의 장황함이 줄어들었고 응답 속도도 이전 대비 절반 수준까지 개선되는 효과를 확인했다.

- 현재는 OpenClaw에 다양한 Skill을 추가로 설치하며, 실제 업무와 개인 활용에 더 편리한 형태로 다듬는 작업을 진행하고 있다.

- 이 과정에서 ClawHub의 문서들이 단순히 사람이 읽기 위한 문서가 아니라, 상당 부분 **AI가 직접 읽고 실행 판단에 활용하도록 설계된 문서**라는 점을 알게 되었다.

- 특히 ClawHub의 skill instruction과 SKILL.md 파일은 AI를 위한 실행 매뉴얼에 가깝다.

- 해당 파일을 특정 폴더에 배치하면, 이후 AI가 필요한 시점마다 이 문서들을 읽고 적절한 CLI 도구와 상호작용하는 방식으로 동작한다.

- 즉, “skill을 설치한다”는 개념은 겉보기와 달리 복잡한 바이너리 설치라기보다, **AI가 참고할 매뉴얼인 Markdown 파일을 정해진 위치에 저장하는 것**에 가깝다.

- 기능 확장 측면에서는 Apple의 iMessage, Notes, Reminder에 접근할 수 있도록 연동을 구성했고, Gmail 연동도 가능하도록 설정했다.
