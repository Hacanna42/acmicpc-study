### IS NULL
어제도 말했듯이, NULL은 알 수 없는 값이므로, !=, = 등의 연산도 알 수 없음으로 평가된다.
따라서, NULL 값을 비교할 때에는 A IS NULL, A IS NOT NULL 식으로 해야 한다.

```sql
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NULL
ORDER BY ANIMAL_ID ASC;
```
